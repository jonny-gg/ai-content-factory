# Story Factory 实现路线

## 已完成
- 故事工厂核心模块 `src/story-factory.ts`
- 单条 CLI `story-cli.ts`
- 批量 CLI `story-batch.ts`
- 导出发布文案、分镜、JSON、音频文件
- 初步商业方案 `BUSINESS-PLAN.md`

## 当前产品形态
输入一个主题，输出：
- 标题
- hook
- 分镜脚本
- 视觉提示词
- 发布文案
- 每个镜头的配音文件

## 下一阶段优先级
1. 接入真实 LLM，而不是当前模板生成
2. 接入 ffmpeg，自动把音频/字幕/图片合成为竖版视频
3. 增加 web 后台，用于选题、批量生成、导出
4. 平台模板细化（抖音/小红书/TikTok）
5. 增加人工审核工作流

## 商业化优先建议
先卖“故事短视频素材包月服务”，后卖 SaaS。
原因：服务最容易验证真实付费意愿。
