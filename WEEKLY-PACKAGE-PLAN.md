# 周更内容包方案

## 目标
一次生成 7 天内容，直接用于连续发布。

## 输入
- 赛道：`horror` / `twist` / `bedtime` / `folktale` / `anime`
- 平台：`douyin` / `xiaohongshu` / `tiktok` / `youtube-shorts` / `bilibili`

## 输出
- weekly-plan.json
- weekly-plan.md
- delivery-note.md
- 7 个 story package 目录

## 商业价值
这个交付形态比单条故事更容易卖，因为客户买的是“连续更新能力”，不是单条脚本。

## 推荐售卖方式
- 每周包：199 元
- 月包（4周）：699 元
- 定制包：999 元起
